# Responsive Dashboard UI

A modern and responsive dashboard interface built using HTML and CSS only.

## Preview

This project features a clean admin dashboard design with:

- Expandable sidebar navigation
- User profile section
- Dashboard statistics cards
- Products table
- Responsive flexbox layout
- Modern dark theme UI
- Font Awesome icons integration

## Features

- Responsive dashboard layout
- Animated sidebar hover effect
- Statistics overview cards
- Product management table
- Clean and organized UI design
- Fully built with pure HTML and CSS

## Technologies Used

- HTML5
- CSS3
- Font Awesome
